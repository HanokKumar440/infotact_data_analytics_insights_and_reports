# Omnichannel Retail Sales & Inventory Analytics Dashboard
This project focuses on analyzing retail sales data collected from multiple locations and sales channels.
By processing and exploring the data, we aim to uncover key sales trends, evaluate product performance, and understand customer purchasing patterns. 
The project builds a complete data analytics pipeline and presents the results through an interactive dashboard to support data-driven business decisions.
